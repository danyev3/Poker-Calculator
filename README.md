# ♠️ ♥️ Poker Calculator ♣️ ♦️

A sleek, fast, and mobile-friendly web app to easily calculate poker payouts at the end of a home game. No more math headaches or endless money transfers between friends—just input the buy-ins and the final cash-outs, and the app will instantly tell you who owes what to whom with the minimum number of transactions.

## ✨ Features
* **Real-time Auto-Calculation:** The results update instantly as you type.
* **Smart Debt Resolution:** Minimizes the number of transactions required to settle up.
* **Auto-Save:** Built-in `localStorage` ensures you don't lose your data if you accidentally close or refresh the page.
* **Multi-Language:** Supports English, Italian, Spanish, French, and German.
* **Glassmorphism UI:** A modern, frosted-glass design tailored for a premium casino vibe.
